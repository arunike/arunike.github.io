Powerpoint slides merging tool

Language written in: Visual Basic for Applications

  • Rating:
  • Link: Github
  • Completed: July 19 2022

This office 360 macro allows the user to merge specific filtered powerpoint together. User is first prompted to a input box, where they could enter a keyword of the powerpoint name that he wish to combine. Then, it will search through the folder and combine all the matching powerpoint. For example: when user enter the word "test", it will filter all powerpoint containing the "test" and merge them together.
